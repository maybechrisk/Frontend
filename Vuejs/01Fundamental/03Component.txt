*뷰 컴포넌트
: 화면 영역을 구분하여 개발하는 뷰 기능 (재사용성을 높이고 개발 속도를 높임)
1.컴포넌트 등록방법 (2가지)
 1) 전역 컴포넌트로 등록
  Vue.component ('app-header',{
    template : '<h1>Header Component</h1>'
  })
 2) 지역 컴포넌트로 등록
  var appHeader = {
    template : '<h1>Header Component</h1>'
  }

* 컴포넌트 통신방식
: 각각 고유한 데이터 유효범위를 가진다.
상위 컴포넌트 -> props 전달 -> 하위 컴포넌트
             <- 이벤트 발생 <-

1.props 속성
: 컴포넌트 간에 데이터를 전달할 수 있는 컴포넌트 통신 방법
